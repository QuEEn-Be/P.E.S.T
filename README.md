## Overview

PEST (Penetration Exploitation & Security Toolkit) is a modular penetration testing toolkit designed to assist with reconnaissance, scanning, and vulnerability discovery during offensive security engagements.

The project is being developed as part of my offensive security training and focuses on building automation for common penetration testing workflows.

The goal is to create a lightweight framework that helps streamline tasks such as:

- Network reconnaissance
- Host discovery
- Port scanning
- Service enumeration
- Vulnerability identification

---

## Project Goals

The primary objectives of this project are:

- Develop a modular penetration testing toolkit
- Automate common reconnaissance and enumeration tasks
- Reinforce Python development skills within a cybersecurity context
- Build practical tooling used during penetration testing labs
